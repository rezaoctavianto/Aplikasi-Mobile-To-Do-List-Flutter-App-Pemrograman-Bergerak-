# Aplikasi-Mobile-To-Do-List-Flutter-App-Pemrograman-Bergerak-
Aplikasi mobile To-Do List adalah aplikasi yang membantu pengguna mencatat, mengelola, dan memantau daftar tugas harian. Pengguna dapat menambahkan tugas, mengedit, menghapus, menandai tugas selesai, serta mengkategorikan tugas berdasarkan prioritas atau tanggal.
